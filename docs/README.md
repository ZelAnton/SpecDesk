- **[design/](design/README.md)** — the concept & architecture docs describing *what* SpecDesk is.

SpecDesk is a Windows desktop app that lets non-technical authors edit Markdown specifications
stored in GitHub without ever touching git, branches, or pull requests directly. See
[design/01-concept.md](design/01-concept.md) for the vision.
