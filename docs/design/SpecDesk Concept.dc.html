A calm, document-first Markdown editor that hides git entirely. All three share the same foundations — light Notion-quiet base (dark mode planned), system UI font, a serif headings + sans body reading surface, one blue accent, Office-balanced density. They differ in how the chrome behaves and how the document sits. Each frame also previews a different live moment so you can judge the full lifecycle.
