# Активная задача

Статус: в работе
Исходная задача: [T-007] Исправить S-08: семантический diff не видит правки чекбоксов, сносок и списков определений

Оценка сложности: высокая
Рекомендуемый effort: high

`src/SpecDesk.Markdown/Projection.fs:14-30, 41-66`. `Pipeline.fs` включает `UseTaskLists`,
`UseFootnotes`, `UseDefinitionLists`, но инлайны `TaskList`, `FootnoteLink`/`FootnoteGroup` и блоки
`DefinitionList` попадают в ветку `| _ -> None` — для них нет отдельных случаев проекции. Из-за этого
переключение `- [ ] x` → `- [x] x` проецируется в одинаковый `ListBlock` → оверлей "Показать изменения"
(единственный потребитель этих данных) сообщает "изменений нет" для реальной, отрендеренной правки;
правка тела сноски или термина/определения также невидима для diff.

Критерии готовности:
- `Projection.fs` получает случаи проекции для инлайнов `TaskList`, `FootnoteLink`/`FootnoteGroup` и
  блоков `DefinitionList` (расширение DU `Ast`, либо отображение состояния задачи на элемент списка) —
  вместо падения в `| _ -> None`.
- Переключение чекбокса задачи детектируется diff'ом как `Changed`, а не как отсутствие изменений.
- Правка тела сноски и термина/определения списка определений детектируется diff'ом.
- Добавлены тесты diff на task-list, сноски и списки определений.
- Запись в `CHANGELOG.md`.
